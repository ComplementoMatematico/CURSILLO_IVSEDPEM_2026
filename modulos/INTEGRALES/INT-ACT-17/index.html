Interpretando área bajo la curva

Objetivo

Reconocer el área bajo una curva como una cantidad con significado: calcularla con geometría elemental, verla como límite de sumas de rectángulos (Riemann), e interpretarla como acumulación. Si $f$ es una tasa, el área es el total acumulado. Es el puente entre geometría e integral.

Función e intervalo
$$f(x)=2x \qquad \text{en } [0,\,4]$$
Panel de parámetros
Recta $f(x) = mx + b$ sobre $[a,b]$. El área bajo una recta es siempre un trapecio (un triángulo si $b=0$). El motor calcula el área exacta y la aproximación por rectángulos.
